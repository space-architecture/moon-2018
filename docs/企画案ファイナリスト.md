# 企画案ファイナリスト

## 電子情報産業案
情報産業の拠点となる施設

- 電子機器製造
- 発電と熱管理の施設
- データセンター
- スーパーコンピュータ
- 地球らとのデータ送受信設備

## 宇宙港案
宇宙船にまつわる様々な業務を担う施設

- 資源採掘と精錬
- 宇宙船の造船と進宙作業
- 燃料補給
- 航路管制
- 船舶法の問い合わせ

## 大規模工場案
地球近辺の宙域における工場

- 採掘した物質から鉱物や酸素の精錬
- 有機廃棄物や原子炉などのリサイクル
- ヘリウム3や超電導や高張力素材などの生産
- 天然や人工の高濃度放射線を利用した放射性廃棄物廃棄や核種変換
